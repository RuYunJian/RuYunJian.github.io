今天是2025.11.23，今天一天和xjz在自习教室度过，下午复习了C语言和CET4还有一些高等数学，

效率并不是很高吧，但是总比坐在宿舍里混日子强吧qvq，晚上回来又决心搞了一下leecode的第1题，

但是我实在搞不懂题目里说的给定只能表示，到底需不需要scanf_s呢，额，有待考究! !

先奉上一段错误示范吧，这是我写的第一版

然后是我觉得的正确示范（先奉上源代码）：

#include <stdio.h>
int main()
{
int a[10];
printf("请输入10个数字");
for (int i = 0; i <= 9; i++)
{
scanf_s("%d", &a[i]);
}
int target = 0;
printf("请输入目标值:");
scanf_s("%d", &target);
for (int j = 0; j < 10; j++)
{
for (int k = 0; k < 10; k++)
{
if ((a[j] + a[k]) == target)
{
printf("%d %d\n", j, k);
break;
}
}
}
return 0;
}
