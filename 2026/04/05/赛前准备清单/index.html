赛前准备清单

2026/4/5 robot

# 机器人竞赛赛前准备文件清单

# 一、Lua 战术脚本文件（.lua）

这些是你必须编写的战术脚本，放在 Team/lua_scripts/play/Ref/ 对应目录下：

文件名 说明 存放目录
NormalPlayDefend.lua 正常比赛战术 play/Nor/
Ref_KickOff.lua 中场开球 play/Ref/KickOff/
Ref_KickOffDef.lua 中场开球防守 play/Ref/KickOffDef/
Ref_BackKick.lua 后场任意球 play/Ref/BackKick/
Ref_BackDef.lua 后场任意球防守 play/Ref/BackDef/
Ref_MiddleKick.lua 中场任意球 play/Ref/MiddleKick/
Ref_MiddleDef.lua 中场任意球防守 play/Ref/MiddleDef/
Ref_FrontKick.lua 前场任意球 play/Ref/FrontKick/
Ref_FrontDef.lua 前场任意球防守 play/Ref/FrontDef/
Ref_CornerKick.lua 角球任意球 play/Ref/CornerKick/
Ref_CornerDef.lua 角球防守 play/Ref/CornerDef/
Ref_PenaltyKick.lua 点球 play/Ref/PenaltyKick/
Ref_PenaltyDef.lua 点球防守 play/Ref/PenaltyDef/
Ref_Stop.lua 停止 play/Ref/GameStop/

# 二、自定义 Skill 技能文件（.dll）

如果你需要自定义技能（比官方13个skill更强），需要用 C++ 编写并编译：

文件名 说明 存放目录
你的技能名.dll 自定义 skill 动态链接库 Team/user_skills/

对应的 C++ 源码文件（编译前）：

文件名 说明
你的技能名.h 头文件
你的技能名.cpp 源文件

# 三、区域调度脚本（.lua，可选修改）

这两个文件已有默认版本，但你可以根据战术需要修改前/中/后场的区域划分参数：

文件名 说明
OurIndirectKick.lua 进攻任意球区域调度
TheirIndirectKick.lua 防守任意球区域调度

# 四、总结

最少准备清单（14个lua + 若干dll）：

├── Ref_KickOff.lua
├── Ref_KickOffDef.lua
├── Ref_BackKick.lua
├── Ref_BackDef.lua
├── Ref_MiddleKick.lua
├── Ref_MiddleDef.lua
├── Ref_FrontKick.lua
├── Ref_FrontDef.lua
├── Ref_CornerKick.lua
├── Ref_CornerDef.lua
├── Ref_PenaltyKick.lua
├── Ref_PenaltyDef.lua
├── Ref_Stop.lua
├── NormalPlayDefend.lua
└── 你的自定义skill.dll（可选，放在user_skills目录）

注意：如果你只用官方的13个task函数（如 task.Shoot()、task.GetBall() 等），就不需要写 .dll 文件。只有当你觉得官方skill不够用、想自己设计更强的技能时，才需要写 C++ 编译成 .dll。
