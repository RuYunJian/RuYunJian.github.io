Cpp笔记总结S1

2026/3/23 note

# 一.string类型

1.求长度：arr.length()或者是arr.size()

2.字符串的连接可以直接使用+号，形如c=a+b（eg：a=hello,b=world,c=a+b=hello world）

3.=和普通变量一样赋值，但是是拷贝的副本

# 二.for循环与auto迭代器

1.auto可以转化为任意数据类型(eg:string a="abc";auto b=a;)

2.定义一个数组arr[]，有如下语法：

int arr[5]={1,2,3,4,5}
for(auto x:arr)
{
    cout<<x<<endl
}

可以依次打印数组中的元素，输出结果就是12345，代替了三分号的步骤

# 三,引用

引用就是取别名 ，具体如下

int a=10;
int &p=a;
P += 5;
cout<<a<<endl;

此时的输出结果为a=15,也就是说P在发生改变时a也会同时发生改变，就是相当于a的别名为p

# 四.常用库函数

I.reverse翻转

假定有一个vector容器储存的arr，则

reverse（arr.begin(),arr.end()）

可以倒转整个vector容器

II.unique去重

作用是去除容器或数组中相邻元素的重复出现的元素，返回值是去重之后的尾地址

int m=unique（arr.begin(),arr.end()）-arr.begin() //m返回的是去重后的元素个数

需要重点注意的是，unique去重的是相邻元素的重复元素，而不是整个容器或数组中的重复元素，所有使用unique的时候一般要对整个容器或者数组进行sort排序（见下文）

III.sort排序

1.sort排序是类似于快速排序的一种排序方式，时间复杂度为n*log2(n),执行效率比较高，一般首选sort

2.sort的使用方法
sort含有三个参数sort（begin,end,cmp）,其中begin为指向待sort（）的数组的首元素的指针，end为指向待sort（）的数组的尾元素的下一位指针（不是尾元素，是尾元素的下一位！！）

cmp：自定义排序函数，返回bool类型，是一个布尔值函数

bool 是 C++ 的布尔类型，只有两个取值：

  • true（真，等价于 1）
  • false（假，等价于 0）

bool 函数的核心作用：返回判断结果（是 / 否、对 / 错）

# ✅ 正确写法

  • 升序：return a < b
  • 降序：return a > b

# ❌ 绝对不能写

  • return a <= b

  • return a >= b

  • bool 函数就是只返回 true 或 false

    a < b 本身，本来就是一个布尔表达式，若a < b，则true，否则返回false
